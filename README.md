# CTF L1 – Entraînement aux commandes UNIX

Bienvenue dans ce **mini CTF (Capture The Flag)** destiné aux étudiants **L1 Informatique**.
L’objectif est de vous entraîner à utiliser les **commandes UNIX/Linux** pour trouver des **flags cachés** dans différents niveaux.

Chaque niveau nécessite d'utiliser votre logique et différentes commandes du terminal.

---

# Installation

Avant de commencer, assurez-vous d’avoir **Git** installé sur votre machine.

### Installer Git

Selon votre système :

**Debian / Ubuntu**

```bash
sudo apt install git
```

**Fedora**

```bash
sudo dnf install git
```

**CentOS / RHEL**

```bash
sudo yum install git
```

**Termux**

```bash
pkg install git
```

---

# Récupérer le CTF

Clonez le dépôt :

```bash
git clone https://github.com/Rafita261/CTF
```

Entrez dans le dossier :

```bash
cd CTF
```

Extraire les challenges :

```bash
tar -xzpf CTF_L1.tar.gz
```

---

# Structure du CTF

Les challenges se trouvent dans :

```
CTF_L1/
```

Il y a **7 niveaux** 

Chaque niveau contient un **flag** à trouver.

---

# Validation des flags

Pour valider un flag, utilisez le **validateur** fourni dans le dossier `CTF`.

Exécutez :

```bash
./validation
```

L’application console vous demandera d’entrer le **flag correspondant à chaque niveau**.

Si le flag est correct, le niveau sera validé.

---

# Objectif

Valider **les 7 niveaux** :

```
niveau0 → niveau6
```
